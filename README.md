# 📚 Study-Group RAG Tutor

A shared study assistant for a whole class. Students join a course space, upload the
course materials once, then ask questions and get **answers grounded in those materials
with citations** — and generate practice quizzes from the same content.

> 🚧 **Early build.** This README is a stub; the full documentation, diagrams, and
> real-run screenshots land in a later phase (see `docs/ROADMAP.md`).
