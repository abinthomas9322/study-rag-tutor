# Demo course data — source & licence

The PDFs in this folder are **real, open-licensed course material** used to seed
the demo `BIO101` course. They are not fabricated; the text is taken verbatim
from the source below. This is explicitly labelled seed/example data (per the
project's data-integrity rules) — it is the genuine content the app indexes and
answers from.

## Source

- **Work:** _Concepts of Biology_, OpenStax
- **Publisher:** OpenStax, Rice University
- **URL:** https://openstax.org/details/books/concepts-biology
- **Full PDF:** https://assets.openstax.org/oscms-prodcms/media/documents/ConceptsofBiology-WEB.pdf

## What's included here

These files are page-range extracts of two chapters from the full book, kept
small so the demo is reproducible without the ~150 MB upstream download:

| File | Source chapter |
|------|----------------|
| `cell-structure-and-function.pdf` | Chapter 3 — Cell Structure and Function |
| `photosynthesis.pdf` | Chapter 5 — Photosynthesis |

The full upstream PDF is cached (git-ignored) under `seed/.cache/` when present;
only these small extracts are committed.
